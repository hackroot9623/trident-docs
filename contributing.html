Contributing

How to Contribute

We greatly appreciate your interest in contributing! If you have suggestions or improvements, please adhere to the following steps to ensure a smooth contribution process:

Contribution Steps

  1. Fork the Repository: This creates your own copy of the repository where you can make your changes.

  2. Create a New Branch: Use the following command to create a new branch where you can make your changes:

git checkout -b feature-branch

  1. Make Your Changes: Update the code or documentation as needed.

  2. Commit Your Changes: Use the following command to commit your changes:

git commit -am 'Describe your changes here'

  1. Push Your Changes: Use the following command to push your changes to your branch:

git push origin feature-branch

  1. Create a new Pull Request: Navigate to the original repository and click on "New pull request". Choose your fork and the branch you worked on, and submit your pull request..
